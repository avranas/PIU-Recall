# PIU Recall
PIU Recall is a score tracking app for the game Pump it Up

## Technologies
1. Node
2. React
3. Express
4. Passport
5. Bcrypt
6. PostgreSQL
7. Sequelize

## Future Work
1. Support for all songs and charts in Pump it Up Phoenix
2. Song search
3. Sort by difficulty level
4. Leaderboards
5. Styling improvements
